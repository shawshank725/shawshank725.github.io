Converting Word to PDF file

Use the following python code to convert Word to PDF file. The required modules need to be installed using pip command.

from docx2pdf import convert

# Converting docx present in the same folder
# as the python file
convert("pps.docx")

# Converting docx specifying both the input 
# and output paths
convert("pps.docx", "pps_converted.pdf")
